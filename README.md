Este repositório é o backend das aplicações do NettStudios.

**Stack de tecnologia**

- **Linguagem:** JavaScript
- **Engine**: NodeJS
- **Framework:** ExpressJS

## Rodando a aplicação

1. Faça o clone deste repositório.

2. Na pasta do projeto, rode o comando de instalação:

   ```shell
   npm i
   ```

2. Depois rode o comando:

   ```shell
   npm start 
   ```

   ou

Rode o comando:

   ```shell
   npm run start:dev
   ```
   

3. Para configurar o arquivo `.env`, é possível utilizar o arquivo de exemplo `.env.example` ou solicitar o arquivo `.env` preenchido para alguém que tenha o devido acesso no projeto.


4. Com o banco de dados e o `.env` configurados, rode a aplicação que desejar (consulte outros comandos no `package.json`):

   ```bash
   npm start
   ```

  <!-- ```bash
   npm run admin:start
   ``` -->

5. Pronto! O backend está disponível em [http://localhost:3000](http://localhost:3000).
